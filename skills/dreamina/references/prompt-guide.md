# Dreamina 提示词编写指南

## 图片提示词

### 基本结构

```
[主体] + [动作/状态] + [环境/背景] + [风格] + [画质修饰词]
```

### 示例

#### 人物类
```
一位年轻女子站在樱花树下，穿着白色连衣裙，微风吹动头发，柔和的光线，电影感，高清，8K
```

#### 动物类
```
一只可爱的橘猫坐在沙发上，毛茸茸的，阳光透过窗户照进来，温馨氛围，超高清，细节丰富
```

#### 场景类
```
赛博朋克风格的未来城市夜景，霓虹灯光，雨后的街道倒影，科幻感，电影质感，4K超清
```

#### 物品类
```
一杯冒着热气的拿铁咖啡，木质桌面，咖啡豆点缀，暖色调，产品摄影风格，高清细节
```

### 风格关键词

| 类别 | 关键词 |
|------|--------|
| 艺术风格 | 油画风格、水彩画、素描、动漫风格、写实风格、抽象艺术 |
| 摄影风格 | 电影感、产品摄影、人像摄影、风光摄影、街头摄影 |
| 氛围 | 温馨、神秘、科幻、复古、清新、梦幻 |
| 光线 | 自然光、逆光、侧光、柔和光线、戏剧性光影 |

### 画质修饰词

```
高清、4K、8K、超高清、细节丰富、高质量、专业级
```

### 负面提示词 (可选)

避免出现的内容：
```
模糊、低质量、变形、水印、文字
```

## 视频提示词

### 运动描述结构

```
[镜头运动] + [主体动作] + [环境变化]
```

### 镜头运动词汇

| 类型 | 描述词 |
|------|--------|
| 推进 | 镜头缓缓推进、向前推进、zoom in |
| 拉远 | 镜头缓缓拉远、向后拉远、zoom out |
| 平移 | 镜头向左/右平移、横向移动 |
| 跟随 | 镜头跟随主体移动、跟拍 |
| 升降 | 镜头缓缓上升/下降、俯拍/仰拍 |
| 环绕 | 镜头环绕主体旋转 |
| 固定 | 固定机位、静止镜头 |

### 主体动作词汇

| 类型 | 描述词 |
|------|--------|
| 人物 | 缓缓走动、转身、微笑、挥手、跑步 |
| 动物 | 奔跑、跳跃、摇尾巴、飞翔、游泳 |
| 自然 | 树叶飘动、云层流动、水波荡漾、花朵绽放 |
| 物体 | 旋转、漂浮、下落、闪烁 |

### 环境变化

```
光影变化、时间流逝、天气变化、昼夜交替
```

### 示例

#### 人物视频
```
镜头缓缓推进，女子微微转身回眸，发丝在风中飘动，阳光穿过树叶形成光斑
```

#### 动物视频
```
猫咪缓缓走动，尾巴轻轻摆动，阳光透过窗户洒在地板上
```

#### 风景视频
```
镜头缓缓向上抬升，云层在天空中缓缓流动，远处的山脉若隐若现
```

#### 过渡视频 (首尾帧)
```
smooth transition between frames
平滑过渡，自然变换
```

## 图生图提示词

### 编辑类型

#### 1. 添加元素
```
添加一顶红色帽子
在背景中添加一轮明月
添加雪花飘落的效果
```

#### 2. 修改元素
```
将背景改为夜景
将头发改成金色
将季节改为秋天
```

#### 3. 去除元素
```
去除背景中的人物
移除画面中的文字
```

#### 4. 风格转换
```
转换为油画风格
转换为动漫风格
转换为复古胶片风格
```

#### 5. 画面微调
```
画面微调
增加对比度
调整色调为暖色
```

## 最佳实践

### 1. 提示词长度
- 简短精炼，30-100 字为宜
- 关键信息前置
- 避免冗余描述

### 2. 语言选择
- 中英文均可
- 英文在某些风格词上效果更好
- 可以中英混用

### 3. 迭代优化
- 先生成后微调
- 根据结果调整关键词
- 保存效果好的提示词模板

### 4. 常见问题

| 问题 | 解决方案 |
|------|----------|
| 细节不足 | 添加"细节丰富"、"高清"等修饰词 |
| 风格不对 | 明确指定风格关键词 |
| 构图不佳 | 添加构图描述如"居中构图"、"三分法" |
| 色调不对 | 指定色调如"暖色调"、"冷色调" |
